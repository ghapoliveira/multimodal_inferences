# Experiment 1 — Norming study

The experiment submitted the 150 sentence-emoji triads (450 sentence-emoji pairs) to public evaluation. It was run on PCIbex and used to select the 120 sentence-emoji pairs most consistently rated as expected for the EEG study (Exp. 2). This directory contains the materials, the web experiment as run, the anonymized responses, and the analysis scripts that produce every table and figure in the Dissertation (Chapter 3).

## Pipeline

```         
corpus.xlsx ──> create_lists.py ──> pcibex_lists.csv ──> PCIbex (pcibex/)
                                                                       │
                                                              responses_ph.csv
                                                                       │
                          Analysis_public_ph.R ──> targets_selected_ph.csv ──> Exp 2
                                                   + norming stats (.rds) + figures
```

`01_corpus_tables.R` describes the corpus (counts, sentence-length summaries), before and after the 120-stimulus selection.

## Reproducing

**Lists (Python):**

``` bash
pip install -r requirements.txt
python create_lists.py
```

The lists used in the experiment were generated unseeded (`RANDOM_SEED = None`), so re-running produces a different but equally valid counterbalanced set. The canonical record of the lists actually presented is the frozen `results/pcibex_lists_asused.csv`.

> Note on `create_lists.py`: a silent bug in the version used to generate the as-used lists prevented the sentence tags from being read, so only the emoji tags constrained the ordering. It was corrected in the final pseudorandomization and in this public script; the visual inspection was the safeguard. The `USE_SENTENCE_TAGS` toggle can produce a version that also uses the sentence tag constraints. 

**Corpus tables and norming analysis (R ≥ 4.1):**

``` r
# Check requirements_R.txt
source("01_corpus_tables.R")        # -> results/tables/
source("Analysis_public_ph.R")      # -> results/analysis_ph/ + .rds tables
```

## Privacy

Only the anonymized data and the `Analysis_public_*.R` scripts are published. The scripts that read raw, identifiable PCIbex output are not part of this repository. The only contacts in the consent screen are the researcher's institutional address and the IESC/UFRJ ethics committee (CEP), both intended to be public.
